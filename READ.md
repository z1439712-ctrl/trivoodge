# CSE-358

## Author
Zahidul Islam

## Description
This program creates a cyan-colored window with my full name as the window title.  
When the user presses the first letter of my name ('Z') from the keyboard, the window closes automatically.

## Language & Tools
- C++
- Graphics Library
- VS Code
- Windows OS

## How to Compile and Run
1. Open the terminal in the project folder  
2. Compile the program: main.cpp  
3. Run the program: main.cpp

## Key Functionality
- Window background color: Cyan  
- Window title: Zahidul Islam  
- Press key *'Z'* to close the window
